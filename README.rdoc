== Welcome to Yeti-Barrage

Yeti-Barrage is a web application used to track the locations of Yetis around the world.
Use Yeti-Barrage to update others and stay up-to-date yourself of yeti movements globally.
Some cities seem to be hit with a barrage of Yetis. Stay frosty.
